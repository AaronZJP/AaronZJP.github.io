iOS 排版设计（Code + Design 翻译）

发表于 2018-11-27

iOS 排版

文本内容占据设计的 80% 以上，因此理解排版是非常关键的。始终要把可读性放在第一位。

字体设计基础知识

学习这些排版以及字体设计的术语和其含义是非常重要的。这些名字将经常用来解释排版以及字体设计。

kerning：字距

ascender：字头

x-height：字高

baseline：基线

descender：字尾

leading：行距

好的排版设计规则

除非你是一个排版爱好者，否则你将会更加的喜欢干净简单的东西，不一定非要是原创的。好的排版让我们想去阅读，所以要去更好的去关注排版的效果。让我们看下这五个好的排版规则，并将之应用到现代化的移动应用中，以及网页上。

旧金山字体

在 iOS 中默认的字体是旧金山字体 San Francisco，它是由 Apple 公司内部制造的。你可以通过这个链接下载新的字体。该字体非常的漂亮以及便于阅读，最重要的是它是免费的。我鼓励你们看下这个视频，以便来理解它是如何影响 iOS 设计的。

旧金山字体跟踪值

iOS会根据字体大小自动调整旧金山的跟踪值和文本/显示。这确保了字体始终易于阅读。在字体处于 20 pt 或者更大的时候，就应该使用 SF UI Display ，或者 SF UI Text

。这些跟踪值只能在Photoshop中应用，但是这里有一个在Sketch中转换的公式。

使用这个 Sketch 插件 可以快速的使用字体的间距。

动态类型

可访问性是 iOS 11 中一个非常重要的一个主题。现在，这里有一些动态的字体预设在下面的图片中。像这样，如果你把文本设置为 Body 并且开启了动态类型后，系统就会基于用户的设置自动的改变字体大小。

正文

选择一个在正文中看起来比较好的字体。当你很疑惑的时候你可以选择一个干净和易于阅读的字体。San Francisco （旧金山字体），Helvetica Neue，Open Sans，Roboto，Proxima Nova 和 Museo Sans 都是几个我比较喜欢的。它们都可以在谷歌字体和 Typekit 上免费获得(通过 Creative Cloud 的账号)。

字体大小

字体大小应至少为 11pt，才能在 iPhone 、iPad 和 Apple Watch 上阅读。虽然这是最小值，但正文文本的推荐大小实际上是 15-19pt。

字大小对与错

你的字体大小不应该低于 10 pt，因为这会让你的许多读者都不适应阅读。

字重

在现代的字体中比如 SF 字体中，除了Regular, and Bold。还有很多额外的字重的：Thin, Ultralight, Light, Medium, Semibold, Heavy and Black 。

一般情况下，字体在 11-19 pt，大小的时候使用 SF Text Regular。字体在 20-34 pt 大小的时候使用 SF 字体显示 Medium 以及在 34pt 或者更大的时候，使用 SF 显示 Bold 。请注意，在更大的尺寸，你可以使用更薄的字体，比如 Light 和 Ultralight ，我建议这样做，如果你想要一个舒适的排版。在iOS 11中，大标题通常要大胆得多。

行高

行高应该在字体大小的 120% 到 145% 之间。

一个正确的例子是行高和字体的大小是相等的（100%）。在下图的左边，我使用了 145% 的比例。这有很大的不同。现在，将这些单词乘以十倍后你就能知道阅读如此拥挤的文本是多么的令人沮丧。

保持每行 45-90 个字符

当你每一行很长的时候，读者就会很难将注意力集中于此。每行大量的文本会让它们疲惫不堪，因为每次跳转到新行都会令人兴奋，而当行太长时，这种情况很少发生。

聪明的使用字体

了解字体是一回事儿，但是怎么去使用字体又是另外一回事儿。漂亮的排版设计是不会凭空出现的，你必须精心策划和定制它。例如，如果字体不支持，不要强制使用粗体或斜体。

使用斜体、下划线、粗体、列表和颜色来加强层次结构和交互。

衬线字体和无衬线字体

无衬线字体是一种在笔触末端没有称为“衬线”的小突出特征的字体。有衬线字体和无衬线字体都是很好的选择。

衬线字体在传统应用程序中更常用，这些应用程序鼓励阅读和讲故事，比如 Medium、iBooks 和《纽约时报》。它很有魅力。就我个人而言，我认为在设计时回归本源很重要，所以我肯定会鼓励在讲故事时使用衬线字体。由于它们在现代应用程序中使用较少，所以您会对它们有一种独特的感觉。

无衬线字体更实用、更中性，在大多数现代应用程序中广泛使用。它的流行也意味着它更通用。无衬线字体的数量是压倒性的，所以要小心选择。

人们说设计不是艺术。不是吗？好的设计才是艺术。

字距

字距是调整两个特定字符之间的间距，以达到令人愉悦的视觉效果结果。

这里有一个很棒的网站，你可以在这里测试你的字距效果。

在哪里去找字体

很多字体都是非常贵的。但是这里有很多免费且很优秀的字体。一定要选你喜欢的字体。这里有一些最好的地方可以找到它们。

Google fonts

在 Google Fonts ，你可以免费的下载你看起来觉得合适的字体，并应用到你的项目中。就免费字体而言，它是最好的选择。由于在iOS上获得自定义字体的唯一方法是下载和导入字体文件，因此下载选项非常有用。至少有两种很棒的字体: Open Sans 和 Roboto，这是现代 Android 手机默认使用的字体。

您可以为 Mac 使用 SkyFonts 自动同步到您的桌面。

如果您想从谷歌字体下载所有可用的字体，有一个 GitHub 仓库 可以实现这一点。

TYPEKIT

Typekitis 是 Creative Cloud 订阅的一部分，拥有丰富的字体选择。对于那些刚刚起步的人来说，它是完美的。我喜欢的字体包括：Proxima Nova, Museo, Adelle Sans and Brandon Grotesque。

FONTSTAND

Fontstand for Macis 最棒的一点是，你可以在购买字体前试用1小时。此外，你不用购买字体，而是以每月价格的一小部分租用它们。

FONTS.COM

对于每月订阅的字体，fonts.com 可以让您使用任意数量的字体。到目前为止，它拥有最大的选择。你可以选择你喜欢的字体。例如，Avenir Next 拥有一个完整的系列，但它并不是免费的。

Fonts.com 还允许您自己托管字体文件，这意味着您可以下载并在服务器上托管它们。因为我对排版很着迷，所以我几乎在所有的项目中都使用这项服务。

资源

版式是一种值得探索的艺术形式。它远远超出了iOS。当你达到可以称之为工艺艺术的水平时，那就是你能获得的最大成就。

排版指南

如果您想了解更多有关组合字体、撇号、引号、破折号和括号的信息，我建议您阅读typogui.de。

更好的排版五分钟指南

一本真正值得一读的书。聪明的设计让你有时间停下来思考每一点。同时也能与很多巧妙的动画效果有所碰撞。

字体配对指南

在某些情况下，您可能希望在设计中使用多种字体来设置正文和标题、标题或块引号之间的对比。本指南特别有助于您根据一组标准对字体进行配对。

手绘字体

还有一门关于手绘字体的有趣课程。使用这些技术，您可以创建徽标、个人品牌和一般的字体设计。

Font Pair.co

字体配对是一个很有用的网站，它帮助您将谷歌字体配对在一起，它有不同的组合类型，如无衬线与衬线，衬线与无衬线，以及许多其他。

Font Base

FontBaseis 字体管理应用程序，有了它，你可以通过将字体添加到任何云文件夹与整个团队共享字体。您还可以在一次单击中访问来自谷歌库的任何字体。这是一个非常有趣的工具，有许多隐藏的特性。

Right Font

通过 Right Font 你也可以与你的团队同步你的字体，可以喜欢任何字体。它自动同步来自谷歌字体和Adobe 、Typekit的字体。你也可以访问不同的图标，你可以在你的设计中使用它们，这是一个非常好的应用。

What The Font

你有没有见过一种字体，想知道是哪种字体？所以这里有一款软件 What The Font ，可以通过上传字体的图片来查找字体，字体会在数据库中找到最匹配的字体。它还有一个iOS版本的应用。
