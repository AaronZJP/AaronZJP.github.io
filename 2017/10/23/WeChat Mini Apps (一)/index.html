微信小程序开发(一)

发表于 2017-10-23

​ 最近公司有个很小的项目，要求用微信小程序来做，于是最近就在学习小程序的相关知识。一直是做 iOS 原生开发，Java Script、HTML 以及 CSS 都不会，微信小程序又和这类知识有关，所有也简单的学习和看了一下这方面的知识。关于前端的这部分会单独的总结成博客记录下学到的东西。这里先简单的说一下微信小程序。

目录结构

​ 微信小程序的开发需要使用微信提供的开发工具 微信开发者工具 来进行开发。首先创建一个小程序的项目，使用 QuickStart 模版创建。该模版创建的项目有两个初始页面，一个小程序页面主要是由以下四个类型的文件组成：

文件类型 是否必须 作用
js 是 页面逻辑
wxml 是 页面结构
wxss 否 页面样式
json 否 页面配置

js 文件主要负责页面的逻辑处理，以及 js 方法的调用等。

wxml 其实就是经过特殊封装等 xml 文件，主要负责页面的布局处理，控件的位置，控件响应、事件绑定的处理

wxss 其实就是 css ，主要负责页面的样式，动画等

json 就是用来存储、保存、读取数据

配置

配置主要是处理微信小程序的主要样式、页面配置、窗口、tabBar、网络请求超时、debug 模式的配置。app.json 文件中的配置是作用于整个微信小程序的，但是是可以被单个页面的 json 配置文件覆盖的，不同的是页面的 json 配置文件只能配置 app.json 文件中的 window 对象，所以可以直接配置 window 对象中的配置项无需使用 window 这个键。

app.json 中所有的配置项如下：

pages：接受一个数组，数组中元素为字符串，改数组需要添加小程序中所有的页面，否则未被加入到数组中的页面是无法被加载的。

window：window的属性如下

属性 类型 默认值 描述
navigationBarBackgroundColor HexColor #000000 导航栏背景颜色，如”#000000”
navigationBarTextStyle String white 导航栏标题颜色，仅支持 black/white
navigationBarTitleText String 导航栏标题文字内容
backgroundColor HexColor #ffffff 窗口的背景色
backgroundTextStyle String dark 下拉背景字体、loading 图的样式，仅支持 dark/light
enablePullDownRefresh Boolean false 是否开启下拉刷新，详见页面相关事件处理函数。
onReachBottomDistance Number 50 页面上拉触底事件触发时距页面底部距离，单位为px
